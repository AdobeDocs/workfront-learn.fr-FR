# Configurer les paramètres globaux du BAT

Commencez par configurer le système de relecture dans [!DNL Workfront] en définissant les paramètres globaux par défaut qui affectent les utilisateurs et utilisatrices. En définissant ces valeurs par défaut dès maintenant, vous vous assurez que les utilisateurs et utilisatrices que vous créez disposent des configurations minimales nécessaires pour commencer à utiliser les fonctionnalités de relecture. Vous gagnez également du temps lorsque vous travaillez sur les configurations de BAT.

* Attribuer des rôles de BAT par défaut
* Établir les paramètres du compte par défaut
* Définir les paramètres de BAT par défaut

Si vous avez déjà commencé à créer des utilisateurs et utilisatrices avant d&#39;établir ces valeurs par défaut, pas de panique. Vous pouvez modifier toutes ces informations dans les profils individuels des utilisateurs et utilisatrices.
