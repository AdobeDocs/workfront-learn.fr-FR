# Installation et utilisation d’un plan directeur

Découvrez comment un administrateur installe un plan directeur et comment un utilisateur le met en œuvre. Dans cet exemple, nous allons installer le plan directeur « Formation Workfront pour les nouveaux utilisateurs » et utiliser les modèles et les vues qu’il crée comme liste de contrôle de formation pour aider les utilisateurs à utiliser Workfront.


>[!VIDEO](https://video.tv.adobe.com/v/3479833/?quality=12&learn=on&enablevpops=0)

## Points clés à retenir

* Comment sélectionner un plan directeur et déterminer s’il a déjà été installé ou non. Cela est possible pour la plupart des utilisateurs disposant d’une licence Standard.
* Comment installer un plan directeur et partager les objets créés par celui-ci. Cela nécessite les services d’un administrateur système.
* Découvrez comment rechercher et utiliser de nouveaux modèles de projet créés en installant un plan directeur.
